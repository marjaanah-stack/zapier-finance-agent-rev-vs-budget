# Notion Database: Monthly Finance Reports

Create a new database (Table view) with these properties:

- **Title** → *Month* (e.g., “2025-09 Revenue vs Budget”) — set as **Title** field
- **Budget Total (GBP)** → Number
- **Actual Total (GBP)** → Number
- **Variance (GBP)** → Number
- **Variance (%)** → Number (format percent)
- **Sales (Budget)** → Number
- **Sales (Actual)** → Number
- **Ops (Budget)** → Number
- **Ops (Actual)** → Number
- **R&D (Budget)** → Number
- **R&D (Actual)** → Number
- **Narrative** → Rich text
- **Slides URL** → URL
- **PPTX URL** → URL

> Tip: You can add properties fast using **Cmd/Ctrl+Enter** to save each, then press **Tab** to move to the next property name field.
